## 主要功能：

自动识别长篇文章，以每行大约32字符宽度进行重新分栏排版，变为 分栏+横向滚动式 阅读，就像小时候读报纸一样。

操作/使用说明：
- `Escape`：如果页面出现错位，可以Escape还原
- `滚动`：滚轮一格半屏、且页面元素自动优先进行横向滚动
- `单击`：滚动对齐阅读当前文章

关键词：快速提升视觉维度 / 刷zhihu神器 / 豪华广角宽屏视角 / 横向滚动阅读模式 / 翻页模式 / 充分利用屏幕空间 / 快阅速读插件 / 雪阅模式
